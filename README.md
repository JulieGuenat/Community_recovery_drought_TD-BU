## Overview

This repository contains all R code used for the statistical analyses and figure generation associated with Guenat et al. We investigated how top-down (apex-predator presence) and bottom-up (nutrient addition) mechanisms shape eukaryotic community recovery dynamics following an experimental drought event across twelve artificial lake ecosystems, using environmental DNA (eDNA) metabarcoding and β-diversity partitioning (turnover and nestedness components).

## Scripts

### `scripts/01_Formatting_env_data.R`
Preprocesses raw physicochemical measurements recorded monthly across twelve lakes. This script only needs to be run if you want to reproduce the environmental data processing from scratch. Steps include:
- Reading raw ODS measurement files (2022 and 2023)
- French-to-English month name mapping
- Averaging measurements across three depths per lake
- Handling of duplicate month names between years (April 2022 vs April 2023)
- Joining with sample metadata
- Output: `data/processed/samples_with_environmental_data.csv`

Note: the output of this script is already provided in `data/processed/` and is a direct input to `02_FULLanalyses.R`.

### `scripts/02_FULLanalyses.R`
Main analysis script. Can be run independently — all required input files are provided in `data/processed/`. Environmental data interpolation to exact eDNA sampling dates, outlier removal, and log-ratio computation of environmental change (δenv) are all performed within this script (Section 6b). Organised into the following sections:

| Section | Content | Output |
|---------|---------|--------|
| 1–2 | Data loading and preparation | — |
| 3–4 | β-diversity across full period (*beta.multi*): GLMMs of treatment effects on turnover, nestedness and total β-diversity | Tables S4–S5, Figure 1, Figure S5 |
| 5–6 | Pairwise β-diversity (*beta.pair*) computation, environmental data integration, interpolation to exact eDNA sampling dates, log-ratio computation of environmental change (δenv) | Figures S2–S4, S6–S7, Tables S2–S3 |
| 7–9 | GLMMs of temporal recovery dynamics with seasonality terms (sine/cosine), residual analyses against δenv variables, within-year models | Figure 2, Tables S6–S9, Figure S8 |
| 10–11 | Between-period and within-period GLMMs, post-hoc pairwise comparisons | Tables S10–S14, Figures 3, S9–S10 |
| 12 | Figure assembly | Figures 2, 3, 4, S9, S10 |

## How to run

### To reproduce all statistical analyses and figures (recommended)
1. Clone this repository
2. Download the eDNA taxonomy table from the companion repository and place it in `data/processed/`
3. Update the `setwd()` path at the top of `02_FULLanalyses.R` to match your local directory
4. Run `scripts/02_FULLanalyses.R`

### To reproduce the environmental data processing from scratch (optional)
1. Download the raw data files from Zenodo and place them in `data/raw/`
2. Update the `setwd()` path at the top of `01_Formatting_env_data.R` to match your local directory
3. Run `scripts/01_Formatting_env_data.R` — this will regenerate `data/processed/samples_with_environmental_data.csv`
4. Then follow the steps above to run the main analysis

## Dependencies

All analyses were run in **R version 4.3.3**. The following packages are required:

```r
# Data handling
tidyverse, lubridate, readODS, readxl

# Statistical analyses
glmmTMB, vegan, betapart, car, emmeans, DHARMa, performance, broom.mixed

# Visualisation
ggplot2, cowplot, ggpubr, ggeffects, ggdist, patchwork, corrplot, sjPlot, effects

# Table output
flextable, officer
```

Install all packages at once:

```r
install.packages(c(
  "tidyverse", "lubridate", "readODS", "readxl",
  "glmmTMB", "vegan", "betapart", "car", "emmeans", "DHARMa", "performance", "broom.mixed",
  "ggplot2", "cowplot", "ggpubr", "ggeffects", "ggdist", "patchwork", "corrplot", "sjPlot", "effects",
  "flextable", "officer"
))
```
