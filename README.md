**Authors:** Julie M. Guenat, Gérard Lacroix, Molly Baur, Céline Stoffel, Nadège Remollino, Stéphane Paolillo, Simon Agostini, Beatriz Decencière, Alexis Millot, Carine Puppo, Gianalberto Losapio & Luca Fumagalli

**Corresponding author:** julie.guenat@ik.me

## Overview

This repository contains all R code used for the statistical analyses and figure generation associated with Guenat et al. We investigated how top-down (apex-predator presence) and bottom-up (nutrient addition) mechanisms shape eukaryotic community recovery dynamics following an experimental drought event across twelve artificial lake ecosystems, using environmental DNA (eDNA) metabarcoding and β-diversity partitioning (turnover and nestedness components).

## Scripts

### `scripts/01_Formatting_env_data.R`
Preprocesses raw physicochemical measurements recorded monthly across twelve lakes. Steps include:
- Reading raw ODS measurement files (2022 and 2023)
- French-to-English month name mapping
- Averaging measurements across three depths per lake
- Removal of one conductivity outlier (values exceeding mean ± 3 SD)
- Linear interpolation of environmental values to exact eDNA sampling dates to account for timing mismatches and one missing month (June 2022)
- Output: `data/processed/samples_with_environmental_data.csv`

### `scripts/02_FULLanalyses.R`
Main analysis script. Must be run after `01_Formatting_env_data.R`. Organised into the following sections:

| Section | Content | Output |
|---------|---------|--------|
| 1–2 | Data loading and preparation | — |
| 3–4 | β-diversity across full period (*beta.multi*): GLMMs of treatment effects on turnover, nestedness and total β-diversity | Tables S4–S5, Figure 1, Figure S5 |
| 5–6 | Pairwise β-diversity (*beta.pair*), environmental data integration, log-ratio computation of environmental change (δenv) | Figures S2–S4, S6–S7, Tables S2–S3 |
| 7–9 | GLMMs of temporal recovery dynamics with seasonality terms (sine/cosine), residual analyses against δenv variables, within-year models | Figure 2, Tables S6–S9, Figure S8 |
| 10–11 | Between-period and within-period GLMMs, post-hoc pairwise comparisons | Tables S10–S14, Figures 3, S9–S10 |
| 12 | Figure assembly | Figures 2, 3, 4, S9, S10 |

## Data availability

The eDNA presence/absence taxonomy table (`1.Euka_DNA_Taxo_verified_pres_abs.csv`) is generated by the companion sequencing data processing pipeline, available at:
[https://github.com/JulieGuenat/Lake_biodiversity_sequencingData_processing](https://github.com/JulieGuenat/Lake_biodiversity_sequencingData_processing)

All raw and processed data files are deposited on Zenodo:
[https://doi.org/10.5281/zenodo.18164452](https://doi.org/10.5281/zenodo.18164452)

## How to run

1. Clone this repository
2. Download the data files from Zenodo and place them in the `data/` directory following the structure above
3. Download the eDNA taxonomy table from the companion repository and place it in `data/processed/`
4. Update the `setwd()` path at the top of each script to match your local directory
5. Run `scripts/01_Formatting_env_data.R` first to generate the processed environmental data
6. Run `scripts/02_FULLanalyses.R` for all statistical analyses and figures

## Dependencies

All analyses were run in **R version 4.3.3**. The following packages are required:

```r
# Data handling
tidyverse, lubridate, readODS, readxl

# Statistical analyses
glmmTMB, vegan, betapart, car, emmeans, DHARMa, performance, broom.mixed

# Visualisation
ggplot2, cowplot, ggpubr, ggeffects, ggdist, patchwork, corrplot, sjPlot, effects

# Table output
flextable, officer
```

Install all packages at once:

```r
install.packages(c(
  "tidyverse", "lubridate", "readODS", "readxl",
  "glmmTMB", "vegan", "betapart", "car", "emmeans", "DHARMa", "performance", "broom.mixed",
  "ggplot2", "cowplot", "ggpubr", "ggeffects", "ggdist", "patchwork", "corrplot", "sjPlot", "effects",
  "flextable", "officer"
))
```
